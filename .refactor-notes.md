## Design notes for client engine

State model:
```ts
type ContextRuntime = {
  id: string;                     // "main" for audio, shapeId for map
  tracks: AudioSourceState[];     // existing per-source loaded/loading/buffer
  isPlaying: boolean;
  selectedAudioUrl: string;       // currently-playing or paused-at track
  duration: number;
  playbackStartTime: number;      // ctx.currentTime when play started
  playbackOffset: number;         // seek-offset when play started
  sourceNode?: AudioBufferSourceNode;
  proximityGain?: GainNode;       // map-room-only; undefined in audio rooms
};

GlobalState {
  contexts: Map<string, ContextRuntime>;
  // legacy top-level fields become selectors over contexts.get("main")
}
```

Migration order:
1. Add `contexts` field; initialize "main"
2. Convert each top-level field that mirrors a context property to a selector
3. Convert each method that takes implicit "this context" to take an optional contextId
4. UI components opt in to a contextId prop (defaults to "main")
